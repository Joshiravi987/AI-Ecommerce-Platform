# AI Ecommerce Platform 🚀

A full-stack AI-powered e-commerce platform built using Spring Boot, React, PostgreSQL, JWT Authentication, and Tailwind CSS.

---

## Features

### Authentication
- User Registration
- User Login
- JWT Authentication

### Product Module
- Add Product
- View Products
- Delete Product
- REST APIs

### Frontend
- React + Vite
- Axios API Integration
- Tailwind CSS UI
- Responsive Dashboard

### Backend
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Layered Architecture
